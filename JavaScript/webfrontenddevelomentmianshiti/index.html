题：请你说一下 Vue 的设计架构。

vue2 采用的是典型的混入式架构，类似于 express 和 jquery，各个部分分模块开发，再通过一个 mixin 去混入到最终暴露到全局的类上。

Vue3 已经发布了，请你说一下与 Vue2 在响应式的实现上有什么区别？

Vue2 采用的是 defineProperty 定义 get 与 set，而 Vue3 改用了 proxy。
