CSS 吸底效果

发表于 2019-11-15 更新于 2019-11-29 分类于 CSS 阅读次数：

今天给大家分享一种网页中常见的一种效果，叫做吸底效果。

当主体区域的内容较少时，页脚区不是随着内容区排布，而是吸底在浏览器的最下方；当主体区域的内容较多，撑出滚动条时，页脚区能随着内容区的增高而离开浏览器的底部，并始终显示在页面的最底部。
